# Version Control Practice

## 📌 Description
This repository is created to practice version control using Git. 
It includes basic operations such as commits, branching, merging, and collaboration workflows.

## 🚀 Objectives
- Learn how to initialize a Git repository
- Practice making commits
- Understand branching and merging
- Resolve merge conflicts
- Work with remote repositories (GitHub)

## 🛠️ Tools Used
- Git.
- GitHub.

## 📂 What I Practised
- git init
- git add
- git commit
- git branch
- git merge
- git push / git pull

## ▶️ How to Use
1. Clone the repository
   git clone https://github.com/your-username/version-control-practice.git

2. Navigate into the folder
   cd version-control-practice

3. Start experimenting with Git commands
